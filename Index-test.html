ℹ️ عن الأداة — Pack Checker
📋 وصف الأداة

أداة التحقق من تعبئة الطلبات — تتيح للموظف سكان الأوردر والتحقق من منتجاته قبل الشحن، مع كتابة ميتافيلد التغليف وتسجيل الحدث في D1.

المستخدمون

موظفو التغليف — يستخدمون السكانر أو الإدخال اليدوي لفحص الأوردرات

كيف تعمل
1سكان باركود الأوردر أو إدخال رقمه → Worker يجلب بيانات الأوردر من Shopify
2الأداة تحدد المرحلة (S1/S2) بناءً على 3 signals: Return نشط + manual_status + status_2_r_e
3لو الأوردر مُغلَّف مسبقاً: يقارن lineItemsEditCount الحالي بالمخزّن في D1 — إذا تغيّر يفتح نافذة Override
4Override: الموظف يختار السبب، يؤكد إرجاع المخزون، ثم يُعيد التغليف
5سكان منتجات الأوردر واحداً بالبـاركود/SKU → عند اكتمالها → زرار "تأكيد اكتمال التعبئة"
6Worker يكتب metafields التغليف في Shopify، يضيف tag، ويسجل في D1
الـ Endpoints الرئيسية
get_orderجلب بيانات الأوردر + تحليل المرحلة + كشف التغيير
complete_packكتابة metafields + tag + تسجيل D1 بالأعمدة الجديدة
get_logs / get_logs_count / get_logs_exportاستعراض وتصدير سجل التغليف
